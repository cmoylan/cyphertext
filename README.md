# Star Commander #

A game.

## Requirements ##

* sdl2
* opengl (3+)
* glew
* libSOIL

### optional requirements ###

* astyle (2.04+)
* cppcheck
* valgrind
* doxygen

## Project Structure ##
```
engine
|--> build (compiled objects and binaries)
|--> doc (documentation)
|--> lib (external dependencies)
|--> src (main code tree)
|--> test (unit tests)
```
